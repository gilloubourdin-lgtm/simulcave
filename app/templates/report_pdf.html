5. Hypothèses

  • Calcul mensuel basé sur la source météo indiquée dans le rapport.
  • La météo dynamique utilise les coordonnées GPS de la cave lorsqu’elles sont disponibles.
  • Si la météo dynamique n’est pas disponible, la région climatique de secours est utilisée.
  • Température effective = température extérieure, température sol ou combinaison des deux selon le type de paroi et le facteur d’enterrement.
  • Formule enveloppe : Q = U × A × ΔT × heures.
  • Les besoins process chaud/froid sont répartis uniquement sur les mois sélectionnés.
  • Les résultats mensuels incluent enveloppe + process.
  • Les parois sont corrigées par le facteur d’inertie du matériau.
